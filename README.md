# CYT(continue your training)

## サイト概要
トレーニング内容と食事の両方を記録し、ユーザ同士で交流が深まるサイト

### サイトテーマ
トレーニング

### テーマを選んだ理由
新型コロナウィルスの影響で在宅率が高くなってきたからこそ、室内で軽い運動やトレーニングを行い、
友人同士で進捗を共有できれば挫折や孤独感なく健康になれると思ったから。

### ターゲットユーザ
20代男女

### 主な利用シーン
休日の自宅やジム
