# CYT(continue your training)

## サイト概要
トレーニング内容と食事の両方を記録し、ユーザ同士で進捗を共有するサイト

### サイトテーマ
トレーニングsns

### テーマを選んだ理由
そもそも筋トレは継続率がとても低く、00人いても１年後に継続できている人は10人も居ない可能性が高いです。
そんな中、新型コロナウィルスの影響で在宅率が高くなってきたからこそ、自宅でトレーニングを行い、
その内容を友人同士で共有できれば挫折や孤独感なく健康になれると思ったからです。

### ターゲットユーザ
20代男女

### 主な利用シーン
休日の自宅やジム、隙間時間
